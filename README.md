# 🍒Cherry Terminal+

Cherry Terminal+ is based on kitty, with H-Linux integration, a new Linux installer, and a compatibility patch.

## Features

- GPU-accelerated rendering
- H-Linux integration
- Simplified version schema
- H-Linux installer integration
- Compatibility patch for existing kitty references
- Improved, upstream kitty performance and architecture

## Purpose

Cherry Terminal+ exists to provide a polished, integrated terminal console experience within the H-Linux ecosystem while remaining closely aligned with upstream kitty.
